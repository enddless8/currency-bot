# currency bot
this discord bot detects currencies in messages and convert them to USD, EUR, RUB, RON

to run the bot:
1. rename "config.json.example" to "config.json"
2. put your token to "config.json"
3. `npm i`
4. you have to get pm2 with `npm i pm2 -g` 
5. `npm run prod`
